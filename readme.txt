An extremely small and simple plugin which allows one to easily embed Flash videos (FLV files) into WordPress blogs.

== Description ==

The Simple FLV plugin (developed by [CyberSEO.net](http://www.cyberseo.net/ "CyberSEO.net")) allows one to easily embed Flash videos (FLV files) into WordPress blogs using the universal FLV tag style:

        [flv:url image width height link player]

where:

* url - URL of the FLV video file you want to embed;
* image - URL of a preview image (shown in display and playlist);
* width - width of an FLV video (default: 450);
* height - height of an FLV video (default: 317);
* link - URL to an external page the display, controlbar and playlist can link to (default: #);
* player - URL to FLV player (default: /wp-content/plugins/simple-flv/flwplayer.swf).

The plugin works with the JW Player 4 by [LongTail Video Inc.](http://www.longtailvideo.com/ "LongTail Video Inc.")

== Installation ==

1. Upload 'simple-flv' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
